Configuration at a Glance

baudRate: integer

Sets the communication speed in bits per second (e.g., 115200, 3000000).

clk_Mhz: integer

Defines the input UART clock frequency in MHz for baud rate generation.

FIFO_DEPTH: integer

Sets the depth of the internal TX/RX FIFOs. Default: 1024.

DEBUG_ON: boolean

Enables the debug ports for FIFO monitoring. Default: false.

Simple Integration

  1. Instantiate IPs

    Add this `uart_dma_driver` and a standard `AXI DMA` to your block design.

  2. Configure DMA

    Set the AXI DMA to Simple Mode (Scatter/Gather disabled) with an 8-bit stream width.

  3. Connect Clocks & Resets

    Wire up the `axis_aclk` to your AXI system clock and `clock` to your UART clock source.

  4. Connect AXI-Streams

    Link the DMA's `M_AXIS_MM2S` to the driver's `s_axis` port and the driver's `m_axis` port to the DMA's `S_AXIS_S2MM`.

  5. Connect Physical Pins

    Route `i_data` (RX) and `data_out` (TX) to your target FPGA pins.
